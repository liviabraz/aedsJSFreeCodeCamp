# Algoritmos e estruturas de dados em JavaScript ✨

<h4> Plataforma: FreeCodeCamp </h4>

Na certificação de Algoritmos e Estruturas de Dados em JavaScript, é ensinado os fundamentos do JavaScript, incluindo variáveis, arrays, objetos, laços e funções.
Depois dos fundamentos, é aplicado esse conhecimento, criando algoritmos para manipular strings, fatorar números e até mesmo calcular a órbita da Estação Espacial Internacional. Durante o processo, também é mostrado dois estilos ou paradigmas de programação importantes: a Programação Orientada a Objetos (POO) e a Programação Funcional (PF).

Neste repositório serão apresentados os projetos feitos para a conclusão do curso. :]

<h2>Tópicos estudados: </h2>

- JavaScript básico
- ES6
- Expressões regulares
- Depuração
- Estruturas de dados básicas
- Programação de algoritmos básicos
- Programação Orientada a Objetos
- Programação Funcional
- Programação de algoritmos intermediários
